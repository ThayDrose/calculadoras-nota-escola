#  Calculadora de Recuperação Anual - Colégio Universo (COC)
Projeto web para alunos do Ensino médio do Colégio Universo. Calcula a Média Anual(MA)
e determina a nota mínima exata (RECANUAL) para aprovação. Utiliza a fórmula de 
pesos do regulamento (MF = (MA x 3 + RECANUAL x 2) / 5) para atingir a Média Final 5.0.
Chega de dúvidas sobre quanto precisa tirar na recuperação anual para ser APROVADO!

##  Como Acessar e Utilizar

Clique no link de acesso direto (URL) do programa.

No campo "Disciplina", insira o nome da matéria.

Nos campos de "Notas Bimestrais", insira suas notas de 0 a 10.

Clique em "Calcular Situação".

O resultado será exibido imediatamente, indicando se você foi aprovado(a) ou qual nota exata precisa na prova de recuperação.

##  Tecnologias Utilizadas

HTML5: Estrutura da página.

CSS3: Estilização e layout responsivo.

JavaScript: Lógica de cálculo e manipulação do DOM.
